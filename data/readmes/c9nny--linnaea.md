 Interactive workbench for inspecting a typed fragment of C through parser output as well as control flow lowering, SSA construction and other obligations. It renders compiler artefacts as Graphviz derived visual models to allow source programs, abstract syntax, proof side conditions and memory events to be studied. 
 
 This project was written primarily to educate femboys and twinks on overall C semantics and tagged unions amongst other things.
