# Driver Monitoring System

面向 Windows 车载视觉研究与教学演示的驾驶员监控及座舱手势交互系统。

> 本仓库仅发布经过打包的 Windows 可执行程序，不包含项目源代码。请从右侧 **Releases** 页面下载最新版本。

[立即下载](DOWNLOAD.md) · [使用指南](docs/USER_GUIDE.md) · [隐私说明](PRIVACY.md) · [更新日志](CHANGELOG.md)

## 核心能力

- **疲劳检测**：基于 EAR、MAR、闭眼持续时间和 PERCLOS 识别持续闭眼、疲劳与打哈欠。
- **分心检测**：融合头部姿态、虹膜位置和持续时间判断低头、转头及视线偏离。
- **驾驶员识别**：支持本地人脸样本录入，并将监控会话与固定驾驶员档案关联。
- **座舱手势 HMI**：支持媒体播放、切歌、音量、前照灯、天窗、空调和电话等模拟指令。
- **数据管理**：使用本地 SQLite 保存会话、指标、危险事件和手势记录，支持筛选及 CSV 导出。
- **隐私与维护**：人脸数据不上传；支持保留周期、定期清理及权限码保护的手动清理。

## 系统要求

- Windows 10/11 64 位
- 可用的 RGB 摄像头
- 建议 8 GB 以上内存
- 建议在光照均匀、面部无遮挡的环境中使用

## 安装与使用

1. 在 **Releases** 页面下载 `DriverMonitoringSystem-v1.0.0-win-x64.exe`。
2. 双击运行；首次启动需要等待程序释放运行组件。
3. 在“驾驶员档案”中建立档案，并在启动摄像头后录入人脸。
4. 回到“实时监控”启动检测。
5. 历史数据、人脸样本和模型保存在 EXE 同目录的 `data` 文件夹。

如 Windows SmartScreen 提示来源未知，请确认文件 SHA-256 与 Release 中公布的校验值一致。本版本尚未进行商业代码签名。

## 手势记忆规则

> 一指管方向，二指管音量，三指管空调，拇指管灯光；张掌播放，握拳暂停。

所有手势均需要持续保持并经过冷却时间确认，以降低误触概率。

## 数据与隐私

本程序默认离线运行。驾驶员档案、监控记录、人脸样本和识别模型均保存在本机，不会由程序主动上传。详细说明见 [PRIVACY.md](PRIVACY.md)。
