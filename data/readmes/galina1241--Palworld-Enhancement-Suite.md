## 🎮 How to Use

### Basic Controls
- **INSERT** or **F1** – Open/Close the enhancement menu
- **Arrow Keys** – Navigate menu
- **ENTER** – Toggle feature ON/OFF
- **ESC** – Close menu
- **HOME** – Minimize suite to system tray
- **END** – Exit suite (all features disabled)

### Quick Hotkeys (Customizable)
- **F2** – Toggle God Mode
- **F3** – Toggle Fly Hack
- **F4** – Infinite Stamina
- **F5** – Instant Capture (when throwing Pal Sphere)

## 🐛 Troubleshooting

| Issue | Solution |
|-------|----------|
| Suite doesn't work | Run as Administrator, disable antivirus |
| Menu doesn't open | Press INSERT or F1, restart both game and suite |
| Game crashes | Reduce graphics settings, disable overlays (Discord/Steam) |
| Features not working | Make sure game is running before opening suite |
| Anti‑cheat triggers | This is for single‑player only – don't use online |
| Pal spawner not working | Make sure you're in the game world, not menu |
| ESP not showing | Enable ESP before entering the game world |
| Speed hack lags | Reduce speed multiplier to 1.5x or 2x |
