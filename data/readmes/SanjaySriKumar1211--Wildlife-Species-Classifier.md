# Wildlife-Species-Classifier

A Flask-based AI application that detects animals from uploaded images using MobileNetV2.

## Features

- Upload animal images
- AI prediction using TensorFlow
- Confidence score
- Animal information
- Flask web interface

## Technologies

- Python
- Flask
- TensorFlow
- MobileNetV2
- HTML
- CSS

## Installation

pip install -r requirements.txt

python app.py
